# 닉네임 기능 설계

## 목적

- 사용자 관점에서의 각 클라이언트 세션을 식별
- 추후 브로드캐스트에서 클라이언트에서 다른 사용자를 식별

## 정책 설계

### 0. 기본 닉네임 설정 정책

1. 기본 닉네임은 해당 `ClientSession`의 `user_(session_id)`로 설정된다.
    - 해당 `session_id`를 기반으로 기본 닉네임을 설정한다면
    유저에게 닉네임을 얻기 전의 정보들과 간단한 로직만으로
    다른 `ClientSession`의 기본 닉네임과 중복이 되지 않는
    해당 `ClientSession`의 닉네임을 설정할 수 있다.

### 1. 커스텀 닉네임 설정 정책

1. 유저가 닉네임을 입력하면 서버에 PacketType == NicknameAssign 타입의 패킷을 보내며 닉네임 설정을 알린다.
2. 서버는 해당 패킷(PacketType == NicknameAssign)을 받았으면 ClientSession의 nickname 필드의 값을 해당 패킷의 페이로드의 값으로 갱신한다.
3. 서버는 클라이언트에 닉네임 설정 완료 패킷을 보낸다.
4. 만약 어떤 이유로든 실패했다면, 연결의 신뢰성이 보장될 경우 닉네임 설정 실패 패킷(NicknameAssignFail)을 보낸다.
    - ($(Current \land (\forall connection (IsReliable(conection)))) \to send(packet)$)

닉네임이 설정되었다면, 앞으로는 클라이언트에서 서버에서 받은 메시지를 출력할 때 닉네임도 함께 표시한다.

### 2. 커스텀 닉네임 변경 정책

1. 클라이언트에서 문자열을 검사한 후 문자열의 시작이 `\nick `이라면 해당 입력은 닉네임 수정 요청으로 간주한다.
2. 그렇다면 클라이언트는 `PacketType::NICKNAME_CHANGE` 타입의 패킷을 보낸다.

### 3. 닉네임 출력 정책

1. 서버에서는 세션 ID 뒤에 표기한다.
```text
[SessionID id][NickName nickname][IP:Port][PacketType] msg
```
2. 클라이언트에서는 이렇게 맨 앞에 표기한다.
```text
[nickname][PacketType] msg
```

## 전체적인 과정 - 경우별로

1. 클라이언트가 처음으로 연결된 경우
    - 디폴드 닉네임으로 해당 `ClientSession`의 닉네임을 설정한다.

2. 클라이언트가 `NICKNAME_CHANGE` 타입의 패킷을 보낸다.
    - 해당 패킷의 페이로드를 `std::string`으로 변형해서 닉네임을 갱신한다.

3. 추가 가능성 있나? `ClientSession::nickname_`가 갱신될만한?

## 수정해야할 부분

1. `ClientSession` `class`
    - `NickName` 필드 추가
    - 생성자에서 해당 클라이언트의 디폴트 닉네임을 설정하는 코드 추가 (전제 : SessionID가 설정되어있어야 함)
    $\forall session (IfGotSessionID(session) \to SetDefaultNickname(session))$
    - 닉네임을 갱신하는 함수 추가
      - 내부에서 `LineLogger` 객체를 통해 로그 출력
    - `Run()` 함수 내에 `PacketType == NickNameAssign`인 패킷의 처리 추가
2. `PacketType` `enum class`
    - `NickNameAssign` 멤버 추가
    - `PacketType::SAFE`를 다른 이름으로 갈어엎기 (예 : `CHAT_MESSAGE`)
3. `LineLogger` `class`
    - 닉네임 생성 과정에서 출력할 로그의 타입 추가

## 구현 예시

### ClientSession 객체

#### 기초 닉네임 설정

```cpp
#include <string>
#include <sstream>

using NickName = std::string;

class ClientSession{
private:
    // ...
    NickName nickname_;
public:
    ClientSession(){
        // ...

        std::ostringstream oss;
        oss << "user_" << session_id;
        nickname_ = oss.str();
    }
};
```
